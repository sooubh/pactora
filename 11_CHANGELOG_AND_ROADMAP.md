# 11. Changelog & Roadmap — Pactora

## Version 1.0.0 (Current)
- Launch of Promises, Borrow, and Money modules.
- Local notification engine.
- Isar-based global search.
- Material 3 Theme support.

## Phase 1 (Next 3 Months)
- **Biometric Lock:** Add Face/Fingerprint unlock for app entry.
- **Custom Categories:** Allow users to create their own promise/item categories.
- **Snooze Presets:** Customizable snooze durations for notifications.
- **Contact Import:** Local-only import of names/phones from device contacts.

## Phase 2 (6 Months)
- **PDF Reports:** Generate professional summaries of lent items and money owed.
- **Voice Notes:** Attach short audio clips to commitments.
- **Home Screen Widgets:** Quick view of today's commitments on the Android home screen.
- **NFC Sharing:** Tap two phones together to "hand over" a promise (Beta).

## Phase 3 (1 Year)
- **Pactora Desktop:** Launch macOS and Windows versions using Flutter Desktop.
- **Local Network Sync:** Sync data between devices on the same Wi-Fi without a cloud server.
- **Trust Score 2.0:** Advanced behavioral analytics for self-improvement.
